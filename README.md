# 📊 Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes retail sales data using Excel and Power BI to uncover key business insights, trends, and performance metrics.

---

## 🛠️ Tools Used
- Excel
- Power BI

---

## 📊 Analysis Included
- Sales trends over time
- Profit analysis
- Regional performance comparison
- Customer segment analysis

---

## 🔍 Key Insights
- Technology category generated the highest profit
- The West region showed the strongest sales performance
